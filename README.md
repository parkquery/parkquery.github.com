### Bower로 설치

Bower가 설치되지 않았다면 다음과 같이 npm을 이용해 Bower를 설치한다.

```bash
$ npm install bower -g
```

Bower를 이용해 dddd를 설치하는 방법은 다음과 같다.

```bash
$ bower install aaaa
```

### npm으로 설치

npm을 이용해 dddd를 설치하는 방법은 다음과 같다.

```bash
$ npm install dddd
```
